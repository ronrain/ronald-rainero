CampusConnect

A multifaceted application empowering college students to enhance their skills within the campus community while gaining valuable entrepreneurial insights. This project includes full CRUD functionality for service listings and service reviews, enabling students to easily update information and enrich their platform experience. The application is built using JavaScript and React.js to deliver dynamic and interactive features. It leverages the MEN stack, where Node.js serves as the server, Express handles routing and middleware, and MongoDB efficiently manages data storage and retrieval, ensuring a seamless user experience.

HealthChat

A dynamic web platform designed to facilitate user-generated discussions on health concerns, empowering individuals to share their experiences, symptoms, and receive valuable feedback from other users. This project encompasses the creation, editing, and management of health discussions, replies, and symptoms through a robust JavaScript CRUD (Create, Read, Update, Delete) system. Real-time user interaction and feedback are enabled through Node.js and Express, enhancing the user experience. Additionally, secure access to user-generated content is ensured through Google Auth and Passport.js authentication. All user information is efficiently stored and managed in a MongoDB database using Mongoose.
